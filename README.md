## 恭喜你！你已经完成了一个拥有以下功能的井字棋啦：

1. tic-tac-toe(三连棋)游戏的所有功能
2. 能够判定玩家何时获胜
3. 能够记录游戏进程
4. 允许玩家查看游戏的历史记录，也可以查看任意一个历史版本的游戏棋盘状态

在这里可以查看最终的游戏代码:[最终成果](https://codepen.io/gaearon/pen/gWWZgR?editors=0010).

- [ ] 6. 当无人获胜时，显示一个平局的消息。
